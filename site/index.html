Cloudflare Worker + Workers AI

Ảnh vào. JSON ra.

Worker mỏng, không khóa nghiệp vụ. Client truyền URL ảnh, prompt, model, tham số và JSON Schema cho từng request. Kết quả giữ rawText, dữ liệu động và chú thích thị giác.

RuntimeCloudflare global network
Vision mặc địnhMoondream 3.1 9B
Triển khaimain → Worker + Pages

API contract

POST /v1/extract
{
  "imageUrl": "https://public.example/error.png",
  "prompt": "Trích xuất thông tin lỗi WinForms...",
  "model": "@cf/moondream/moondream3.1-9B-A2B",
  "parameters": { "max_tokens": 1800, "temperature": 0 },
  "output": { "includeRawText": true, "schema": { "type": "object" } }
}

Pipeline tối giản

01ValidateAuth, URL, model
02FetchHTTPS public image
03InferWorkers AI binding
04NormalizeJSON ổn định
Nguyên tắc: Worker không chứa field nghiệp vụ. Field thuộc prompt và schema do caller gửi.
